# 🏋️ FitnessClub CRM

**Система управления фитнес-клубом / студией / салоном красоты**  
(адаптируется под любой бизнес с абонементами и посещениями)

---

## 🎯 Для чего это?

Фитнес-клубы, студии йоги, парикмахерские, стоматологии и другие сервисные бизнесы теряют клиентов из‑за хаоса в учёте.  
FitnessClub CRM решает:

- ✅ учёт клиентов и абонементов  
- ✅ отметка посещений (админ и клиент)  
- ✅ продажа абонементов через админа  
- ✅ статистика и отчёты  
- ✅ личный кабинет клиента  

---

## 👥 Роли и возможности

### 🛡️ Администратор (владелец / менеджер)

| ID | Сценарий | Что делает система |
|----|----------|-------------------|
| UC‑1 | **Добавление нового клиента** | Создаёт учётную запись (ФИО, телефон, пароль, email) → можно сразу добавить абонемент |
| UC‑2 | **Продажа абонемента** | Ищет клиента → выбирает тип (разовый / месячный / квартальный / годовой / на 8 или 12 посещений) → указывает цену → система активирует и показывает дату окончания / остаток посещений |
| UC‑3 | **Отметка посещения** | Ищет клиента → нажимает «Отметить» → система проверяет активность абонемента и дубли за сегодня → фиксирует посещение, списывает день / раз |
| UC‑4 | **Просмотр статистики** | Выбирает период (день / неделя / месяц / произвольный) → видит: общее число посещений, доход от продаж, графики посещаемости, топ активных клиентов |

---

### 👤 Клиент (посетитель)

| ID | Сценарий | Что делает система |
|----|----------|-------------------|
| UC‑5 | **Просмотр своего абонемента** | Показывает тип, дату окончания, остаток посещений, статус (активен / истёк) |
| UC‑6 | **Просмотр истории посещений** | Показывает список дат и времени всех посещений |
| UC‑7 | **Заказ перезвона менеджера** | На главной странице → кнопка «Заказать звонок» → модальное окно (Имя, телефон) → система отправляет заявку админу |

---

## 🛠️ Технологии

- **Backend:** ASP.NET Core 8, Entity Framework Core, JWT  
- **Database:** SQLite / PostgreSQL (на выбор)  
- **Frontend:** HTML5, CSS3, Vanilla JS  
- **Deploy:** Docker + любой VPS (Ubuntu)
